## 项目背景

OneID 是一个生产就绪的企业级身份认证管理系统，解决企业 SSO、微服务身份网关、SaaS 多租户应用等场景下的统一认证问题。

## 核心特性

- **标准合规** — 完全符合 OAuth 2.1 和 OpenID Connect 标准协议
- **企业 SSO** — 支持跨系统单点登录，统一身份管理
- **多租户支持** — 适用于 SaaS 多租户架构，租户间数据完全隔离
- **安全特性** — 内置 MFA、令牌轮换、审计日志等安全机制
- **监控分析** — 提供丰富的登录行为分析与实时监控仪表盘
- **管理界面** — 现代化 Web 管理后台，支持用户、角色、权限的可视化管理

## 技术栈

基于 ASP.NET Core 构建，采用领域驱动设计（DDD）模式，支持多种数据库后端，可容器化部署。
